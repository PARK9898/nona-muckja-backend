# 노나먹자 백엔드 REST API

## 프로젝트 소개

소비자의 배달비 부담을 절감하기 위한 배달음식 공동구매 소셜링 플랫폼 노나먹자

- [프론트엔드 Repository](https://github.com/clroot/nona-muckja-frontend)

## 프로젝트 구조

```
/backend
├── config
├── domain
│   ├── party
│   └── user
├── exception
├── security
│   ├── jwt
│   └── service
├── service
└── web
    ├── controller
    └── dto

```
 ## 사용 기술
- JAVA (JDK17)
- Spring Boot (v2.6.6) 
- Spring Data JPA
- Spring Security
- queryDsl
